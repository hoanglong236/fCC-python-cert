# User Configuration Manager

This is a Python-based utility designed to handle user settings. It allows for adding, updating, deleting, and viewing user settings efficiently.

## Features
- **Add/Update Settings**: Dynamically modify user preferences.
- **Validation**: Ensures configuration keys meet specific criteria.
- **Formatted Output**: Displays settings in a clean, readable string format.

## How to Run
To use this manager, run:
```bash
python main.py
```

To test this manager, run:
```bash
python test.py
```

## Tech Stack
- **Language:** Python 3.14.0
- **Testing:** Unit Testing (AI-assisted with GitHub Copilot)
